Welcome to DynamicMapper

DynamicMapper is a library used for transforming input objects into custom output objects through a set of actions which map properties from input object type into output object type.

Latest Version

The quickest way to get the latest release of DynamicMapper is to add it to your project using NuGet (http://nuget.org/List/Packages/DynamicMapper).
